# ▶️ Getting Started

## Clone Repository

```bash
git clone https://github.com/Almase02/Full-Stack-Web-Development.git
```

## Install Dependencies

```bash
npm install
```

## Run React

```bash
npm start
```

or

```bash
npm run dev
```

## Run Backend

```bash
node server.js
```
